# Smart Campus Resource Management System

## Overview
Smart Campus Resource Management System is a web-based application designed to efficiently manage campus resources such as laboratories, classrooms, equipment, and bookings.  
The system helps users easily check availability, reserve resources, and manage bookings through a simple and modern interface.
